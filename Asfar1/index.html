Momentum Drop Lab

Drop 3 balls from the same height. They hit the ground with the same speed (ignoring air resistance), but different momentum because p = m·v.

Controls

Gravity \(g\) is set to m/s². Air resistance is ignored.

Bounce rule: on impact, velocity becomes v′ = −e·v.

Impact values (theory)

Theory uses \(v = \sqrt{2gh}\). At the same height, all balls have the same \(v\), so the momentum difference comes from mass.
