Momentum Lab

This page focuses on momentum at ground impact. With no air resistance, dropping from the same height gives the same impact speed, so momentum differs only because mass differs.

Controls

Physics rule: the only acceleration is gravity.

p = m · v momentum at impact
v = √(2 g h) drop from rest
h = v² / (2 g) bounce height from bounce speed

Why velocity can be the same: for a drop from rest, \(v=\sqrt{2gh}\) depends on height and gravity only.
Why momentum differs: momentum uses mass too, \(p=m·v\), so heavier balls have larger momentum at the same speed.

Simulation

Quality checks to try: change mass only (impact speed should not change), change height (impact speed should change), change gravity (impact speed + bounce height should change), verify p = m·v numerically.
